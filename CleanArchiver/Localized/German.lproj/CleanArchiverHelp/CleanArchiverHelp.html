CleanArchiver Hilfe

Anleitung

Ziehen Sie die gewünschten Dateien oder Ordner auf das Icon des Programms. CleanArchiver wird aus diesen Objekten ein Archiv erzeugen.

Sie können einige Einstellungen im Hauptfenster setzen. Ein Klick auf "Speichern als Standard" sichert diese von Ihnen gesetzten Optionen.

CleanArchiver entpackt keine Archive. Verwenden Sie dafür z.B. StuffIt Expander.

Archivierungs-Optionen

CleanArchiver bietet einige Optionen, z.B. den Archiv-Typ.

Archiv-Typ

Sie können unter verschiedenen Archiv-Arten wählen: gzip, bzip2, zip oder Disk Image.

Sichere Resource Fork

Diese Option ist nur bei den Archiv-Typen gzip und bzip2 verfügbar. Im Resource Fork einer Datei werden Mac-spezifische Informationen gespeichert. Unter anderen Betriebssystemen werden diese Infos nicht benötigt.

Ohne .DS_Store-Dateien

.DS_Store-Dateien sollen nicht ins Archiv übernommen werden.

Ohne Icon-Dateien

Die Icon-Datei "Icon\r", die vom System erstellt wird, wenn Sie einer Datei oder einem Ordner ein eigenes Icon zuweisen, soll nicht ins Archiv übernommen werden.

Einzeln archivieren

Ist diese Option aktiviert, dann wird CleanArchiver pro Objekt (Datei oder Ordner) ein eigenes Archiv erstellen anstatt die Objekte in einem Archiv zusammenzufassen.

Automatisch ersetzen

Ist diese Option aktiviert, dann wird das Programm eine bestehende gleichnamige Datei im Zielordner ohne Nachfragen ersetzen.

Internet-fähiges Disk Image

Diese Option ist nur bei Ordnern verfügbar, wenn der Archiv-Typ "Disk Image" ist. In diesem Fall wird ein "Internet-Enabled Disk Image" erzeugt, das nach dem Öffnen direkt auf dem Zielrechner als Ordner entpackt wird.
