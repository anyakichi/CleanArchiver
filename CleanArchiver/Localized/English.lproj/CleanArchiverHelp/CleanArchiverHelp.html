CleanArchiver Help

Usage

Drag & drop the target files and/or folders to the application's icon or window, and it outputs a complessed archive.

You can select some options in the main window. If you push "Save as Default", these options are saved and used as the standard options.

CleanArchiver doesn't unarchive. Use StuffIt Expander instead.

Archiving Options

CleanArchiver has some options including archive type.

Archive Type

You can select an archve type from among gzip, bzip2, zip, Disk Image, StuffIt, and StuffIt X. If you want to archive with StuffIt or StuffIt X, you need to install the DropStuff application on your system.

Exclude .DS_Store

When on, .DS_Store files are not included in the archive.

Exclude Icon File

When on, the custom icon file "Icon\r" that is created when you set an icon for a file or folder is not included in the archive.

Archive Individually

When on, when multiple items are dropped, CleanArchiver archives them individually instead of grouping them.

Replace Automatically

When on, even if an archive file name decided by the application already exists, the application replaces an old file with the new file without any caution.

Internet-Enabled Disk Image

It is used only when Archive Type is "Disk Image." When on, Internet-Enabled Disk Image is created.
