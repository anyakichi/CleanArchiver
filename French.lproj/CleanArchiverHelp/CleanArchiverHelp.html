Aide CleanArchiver

Utilisation

Glisser-déposer les fichiers et/ou les dossiers cible sur l'icône de l'application ou dans la fenêtre, et elle produit une archives comprimée.

Vous pouvez choisir quelques options dans la fenêtre principale. Si vous enfoncez le bouton "Enregistrer par défaut", ces options sont sauvegardées et utilisées comme options standard.

CleanArchiver ne désarchive pas. Utilisez StuffitExpander à la place.

Options d'archivage

CleanArchiver a quelques options comprenant le type d'archives.

Type d'archives

Vous pouvez choisir un type d'archive à partir les formats gzip, bzip2, zip, image disque, Stuffit, et Stuffit X. Si vous voulez archiver avec Stuffit ou Stuffit X, vous devez installer l'application de DropStuff sur votre système.

Exclure les .DS_Store

Lorsqu'en fonction, les fichiers .DS_Store ne sont pas inclus dans les archives.

Exclure les fichiers d'icônes

Lorsqu'en fonction, le fichier d'icône personnalisé "Icon\r" qui est créé quand vous définissez une icône pour un fichier ou le dossier n'est pas inclus dans les archives.

Archiver individuellement

Lorsqu'en fonction, lorsque de multiples éléments sont déposés, CleanArchiver les archive individuellement au lieu de les grouper.

Remplacer automatiquement

Lorsqu'en fonction, même si un nom de fichier d'archive décidé par l'application existe déjà, l'application remplace l'ancien fichier avec le nouveau fichier sans aucun avertissement.

Image disque adaptée à Internet

Elle est utilisée seulement lorsque le type d'archives est "Image disque." Lorsqu'en en fonction, une image disque adaptée à Internet est créée.
